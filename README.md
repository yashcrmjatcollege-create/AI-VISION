# AI-VISION
AI VISION is a multi-LLM document processing platform built with Streamlit and DSPy. It provides AI-assisted document editing, summarization, feedback generation, and visualization features while supporting multiple AI models through OpenRouter, Gemini, and local Ollama integrations.
